# 📚 教案輔助評論系統

一個基於 NotebookLM 介面設計理念的教案評論與分析系統，採用 Google Material Design 深紫色主題配色。

## 🎨 設計特色

### 介面架構

參考 NotebookLM 的三欄式布局設計：

- **左側面板**：教案來源管理（類似 NotebookLM 的 Sources Panel）
- **中間面板**：AI 對話互動（類似 NotebookLM 的 Chat Panel）
- **右側面板**：記事、評分與生成內容（類似 NotebookLM 的 Studio Panel）

### 配色方案

基於 **Google Material Design**，採用深紫色調動態配色：

- **主色調**：深紫色 (#6A1B9A) - 專業且具教育氛圍
- **強調色**：琥珀色 (#FFA726) - 用於評分星號
- **輔助色**：青綠色 (#00796B) - 用於成功提示
- **背景色**：淺灰與白色系 - 保持閱讀舒適度

## 🚀 快速開始

### 安裝步驟

1. 將所有檔案放置在同一目錄下：

   ```
   教案評論系統/
   ├── index.html      # 主頁面
   ├── styles.css      # 樣式表
   └── app.js          # JavaScript 功能
   ```

2. 直接在瀏覽器中開啟 `index.html`

### 使用方式

1. **上傳教案**：點擊「新增教案」按鈕上傳文件
2. **選擇教案**：點擊教案項目進行選擇（支援多選）
3. **AI 對話**：在對話框中輸入問題或點擊建議操作
4. **評分教案**：切換至「評分」標籤頁進行評分
5. **生成內容**：在「生成內容」標籤頁產出各類分析報告

## 🔧 擴展開發

### 後端整合建議

```javascript
// API 端點設計
POST   /api/lessons          // 上傳教案
GET    /api/lessons          // 獲取教案列表
GET    /api/lessons/:id      // 獲取單一教案
POST   /api/chat             // AI 對話
POST   /api/scores           // 提交評分
POST   /api/generate         // 生成內容
```

### AI 模型整合

建議整合以下 AI 服務：

- **OpenAI GPT-4**：對話與文本生成
- **Google PaLM 2**：教育領域專業分析
- **Anthropic Claude**：長文檔分析

### 資料庫設計

```sql
-- 教案表
CREATE TABLE lessons (
    id INT PRIMARY KEY,
    title VARCHAR(255),
    file_path TEXT,
    upload_date TIMESTAMP,
    user_id INT
);

-- 評分表
CREATE TABLE scores (
    id INT PRIMARY KEY,
    lesson_id INT,
    objectives FLOAT,
    content FLOAT,
    innovation FLOAT,
    assessment FLOAT,
    timing FLOAT,
    comment TEXT,
    created_at TIMESTAMP
);

-- 記事表
CREATE TABLE notes (
    id INT PRIMARY KEY,
    lesson_id INT,
    content TEXT,
    created_at TIMESTAMP
);
```
